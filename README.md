# Samurai DoJo (サムライ道場)

## 📌 アプリの趣旨・コンセプト
「Samurai DoJo」は、Webアプリの力を使って、「素振り（Suburi）」や反復動作を最高のエンターテインメントに変えるWebアプリケーションです。
剣道の木刀（Bokken）や素手でのスイング動作をAIがリアルタイムに骨格検知し、自動でカウント。単なる回数カウントにとどまらず、エフェクトや効果音、さらには「モンスター討伐モード」といったゲーミフィケーション要素を取り入れることで、退屈になりがちな反復練習を楽しく習慣化できるように設計されています。

## ✨ 主な機能と特徴
1. **骨格検知を利用した自動カウント (MediaPipe)**
   - PCやスマートフォンのカメラを使用し、ユーザーの肩・肘・手首の動きをリアルタイムにトラッキング。
   - 映像データはサーバーに送信されず、すべてブラウザ内（エッジ側）で完結するため、安全かつ非常に高速に動作します。
2. **モンスター討伐モード（ゲーミフィケーション）**
   - 任意の画像をアップロードし、目標回数（HP）を設定。
   - 素振りを行うたびに、画面上の「モンスター」の画像にリアルなひび割れが入り、最後は画面が派手に割れる演出とともに討伐完了を祝います。
3. **超低遅延のプロシージャルサウンド (Web Audio API)**
   - 外部の音声ファイルをダウンロードするのではなく、ブラウザ上で直接音を合成。振り下ろした際の「風切り音」や、画像にヒビが入る「衝撃音」を遅延ゼロで再生・合成します。
4. **PWA（Progressive Web App）対応**
   - スマートフォンの「ホーム画面に追加」機能に対応しており、まるでネイティブアプリのように全画面かつ高速で起動可能です。
5. **柔軟なカスタマイズ (カメラ選択・設定調整)**
   - 外付けのWebカメラへの切り替え、判定に使用する利き手の設定、カウントの感度（速度閾値）やクールダウン時間の調整など、ユーザーの環境に合わせたセッティングが行えます。

## 🎯 使い方（User Guide）
1. **準備**
   - アプリを開き、カメラへのアクセスを許可します。
   - スマートフォンやPCのカメラの前に立ち、上半身（特に両手と肩）が画面に収まるように調整してください。（設定メニューから使用するカメラの切り替えも可能です）。
2. **通常カウントモード**
   - スマホから離れて、木刀（または素手）を頭上まで大きく振りかぶり、勢いよく腰の高さまで振り下ろします。
   - 画面のUIが状態に応じて変化（待機中→振り上げ検知→振り下ろし中→カウント成功）し、効果音とともにカウントが加算されます。
3. **モンスター討伐モード**
   - 画面下部の討伐モードパネルから、好きな画像（倒したい敵、達成したい目標、苦手な野菜の画像などなんでもOK）をアップロードします。
   - 目標回数（HP）を設定し、「討伐開始！」を押します。あとは全力で素振りをして対象を粉砕するだけです。

## 🎤 プレゼン構成用の参考台本メモ（Pitch / Presentation Notes）

**【1. 現状の課題（フック）】**
「皆さん、毎日の素振りやエクササイズ……ちゃんと続いていますか？」
一人で黙々と数を数えるだけの反復練習は、モチベーションを維持するのが最も難しいタスクの一つです。

**【2. 解決策：Samurai DoJo】**
そこでプロトタイプから発展し、開発したのが、デバイスのカメラとAIを組み合わせた「Samurai DoJo」です！
専用センサーの購入や、面倒なアプリのインストールは一切不要。URLにアクセスしてブラウザを開くだけで、あなたの部屋が瞬時に「道場」に変わります。

**【3. デモ実演の見どころポイント（アピール）】**
- **「プライバシー＆スピード」**: 注目して頂きたいのは反応速度です。映像はクラウド上のサーバーに送られず、すべて手元のブラウザのAI（Google MediaPipe）で処理されています。だからこそノータイムで判定でき、プライバシー上も極めて安全です。
- **「圧倒的な爽快感」**: Web Audio APIで作られた遅延ゼロの「シュッ！」という風切り音と、モンスター討伐モードの「パリンッ！」とガラスが割れるエフェクトをぜひ体験してください。
- **「楽しさの拡張性（ユーモア）」**: 単なるトレーニングツールではなく、「好きな画像を敵に見立てて叩き斬る」というユーモアを取り入れました。今日のプレゼンの緊張を吹き飛ばすため、プレッシャーを与えてくる敵の画像をセットするのも良いですね！

**【4. 今後の展望】**
現状はスタンドアローンのPWAとして完成していますが、今後はFirebaseのようなバックエンドと連携し、「日々の練習データのグラフ化」や、「世界中のSamuraiたちとのオンライン討伐ランキング（非同期マルチプレイ）」などに発展させるポテンシャルを秘めています。

## 🛠 技術スタック (Technical Stack)
- **Frontend Framework**: React (TypeScript) + Vite
- **Styling**: Tailwind CSS
- **AI / Pose Estimation**: MediaPipe Tasks Vision (`@mediapipe/tasks-vision`)
- **Sound**: Web Audio API (Procedural real-time synthesis)
- **Icons**: Lucide React
- **Deployment**: Vercel (推奨)
