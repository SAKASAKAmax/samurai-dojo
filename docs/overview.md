# SAMURAI DoJo — 機能一覧 & 使い方

> リアルタイムポーズ推定を使った木刀素振りカウンター

---

## このアプリについて

剣道における素振りの回数を自動でカウントし、己を鍛えるために作られたアプリです。カメラの前で木刀を振るだけで回数が記録されるため、一人での稽古に集中できます。

---

## 機能一覧

### コア機能

| 機能 | 概要 |
|------|------|
| **自動カウント** | カメラ映像をリアルタイム解析し、振りかぶり→振り下ろしを1回として自動カウント |
| **ポーズ推定** | MediaPipe Pose Landmarker で肩・肘・手首の位置を毎フレーム検出 |
| **カウント表示** | カメラ映像上に大きくカウント数をオーバーレイ表示 |
| **状態表示** | 待機中 / 振り上げ検知 / 振り下ろし中 / カウント成功 の4ステートをリアルタイム表示 |

### モンスター討伐モード

| 機能 | 概要 |
|------|------|
| **好きな画像を標的に** | 任意の画像ファイル（PNG / JPEG）をアップロードして討伐対象に設定 |
| **HP設定** | 目標回数（HP）を自由に設定。振るたびにHPが減っていく |
| **クラックエフェクト** | ダメージが蓄積するほど画像にひび割れが入るビジュアルエフェクト |
| **ヒットエフェクト** | 振るたびに振動＆フラッシュのSEとアニメーション演出 |
| **討伐完了画面** | 目標回数到達で記念画面を表示 |

### カスタマイズ設定

| 設定項目 | 内容 | デフォルト |
|----------|------|-----------|
| **カメラ選択** | 複数カメラがある場合に使用するカメラを選択 | デフォルト（前面推奨） |
| **判定する手** | 左手 / 右手 / 両手平均 から選択 | 両手平均 |
| **感度** | 振りの速度閾値（0.1〜1.9）。高いほどゆっくりでも判定 | 0.5 |
| **クールダウン時間** | 1回カウント後の待機時間（300〜2000ms）。連続誤検知を防止 | 800ms |
| **デバッグ表示** | FPS・手首位置・速度などの推定データを表示 | ON |

### レスポンシブレイアウト

| デバイス | レイアウト |
|----------|-----------|
| **PC** | 左：カメラ映像 / 右：ダッシュボード + 討伐モード（討伐中は左右入れ替わり） |
| **タブレット / スマホ** | 縦積みレイアウト |

## 基本的な使い方

### 1. アクセス

ブラウザで以下のURLを開く（カメラ・マイクのアクセス許可を求められたら許可する）

```
https://samurai-dojo-ten.vercel.app
```

### 2. 素振りカウントを始める

1. カメラの前に立ち、**上半身がフレームに収まるように**距離を調整する
2. 木刀（または素手）を持ち、構える
3. **振りかぶってから振り下ろす** → 自動でカウントされる
4. カウントをリセットしたい場合は **「リセット」** ボタンを押す

### 3. モンスター討伐モードを使う

1. 画面下部（PCは右側）の **「モンスター討伐モード」** エリアで討伐したい画像をアップロード
2. 目標回数（HP）を設定する
3. **「討伐開始！」** ボタンを押す
4. あとは振るだけ — HPがゼロになると討伐完了

### 4. 設定を調整する

「設定」ボタンから以下を変更できる

- カメラが複数ある場合は使用するカメラを選択
- 判定精度が合わない場合は **感度** を調整
- 二重カウントが気になる場合は **クールダウン時間** を長くする

## 動作要件

- モダンブラウザ（Chrome / Safari / Edge 推奨）
- カメラ（内蔵・外付け問わず）
- 上半身が映る程度の空間
- インターネット接続（初回起動時に必要な機能を読み込むため）
