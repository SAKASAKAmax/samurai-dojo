# 木刀素振り自動カウントアプリ POC/MVP ドキュメント

## 1. 実装計画
- **Phase 1 (現在):** PCブラウザで動作するPOCの作成。カメラ入力、MediaPipeによる骨格推定、状態遷移ベースの素振りカウントロジックの実装。デバッグUIの提供。
- **Phase 2:** UIの整理、設定の永続化、セッション管理の追加。
- **Phase 3:** モバイル対応（レスポンシブ化、カメラの前後切り替えなど）。

## 2. 技術選定理由
- **フロントエンド:** React + TypeScript + Vite (高速な開発サイクル、型安全、コンポーネント指向)
- **スタイリング:** Tailwind CSS (ユーティリティクラスによる迅速なUI構築)
- **姿勢推定:** `@mediapipe/tasks-vision` (Pose Landmarker)
  - ブラウザ上でリアルタイムかつ高精度に動作する。
  - MoveNetと比較して、全身のランドマーク（33点）が安定して取得でき、Z軸（奥行き）の情報も得られるため、素振りのようなダイナミックな動きの判定に適している。

## 3. 画面遷移図
```text
[ 初期画面 (カメラ許可待ち) ]
       | (許可)
       v
[ メイン画面 (プレビュー・カウント表示) ]
       |
       +-- (設定ボタン) --> [ 設定モーダル (感度・利き手など) ]
```

## 4. 状態遷移図 (素振り判定ロジック)
```text
       (開始)
         |
         v
    [ IDLE (待機) ] <-----------------------------------------+
         |                                                    |
         | (手首が肩/頭より上に移動 & 上方向への速度検知)     |
         v                                                    |
   [ RAISED (振り上げ) ]                                      |
         |                                                    |
         | (手首が下方向へ一定速度以上で移動)                 |
         v                                                    |
 [ DOWNSWING (振り下ろし) ]                                   |
         |                                                    |
         | (手首が胸/腹の高さまで下降)                        |
         v                                                    |
   [ COUNTED (カウント) ] --(カウント+1)--> [ COOLDOWN ] -----+
                                              (一定時間経過)
```

## 5. ローカル起動手順
本環境（AI Studio）では自動的に起動・プレビューされます。
ローカル環境で実行する場合は以下のコマンドを使用します：
```bash
npm install
npm run dev
```

## 6. 既知の課題一覧
- ユーザーの身長やカメラの角度によって、手首の相対的な高さが変わるため、固定の閾値では判定が漏れる・誤爆する可能性がある。
- 高速すぎる素振りの場合、Webカメラのフレームレート（通常30fps）では中間の動きが欠落し、状態遷移がうまくフックしないことがある。
- 複数人がカメラに映り込んだ場合の挙動が未定義（現在は最初に検出された人物を対象とする）。

## 7. 今後の改善候補
- **キャリブレーション機能:** ユーザーに「構え」「振り上げ」「振り下ろし」のポーズをとってもらい、個人の骨格サイズやカメラ画角に合わせた閾値を自動計算する。
- **平滑化処理の強化:** カルマンフィルタや移動平均を用いて、ランドマークのブレを軽減する。
- **PWA化:** モバイル端末でネイティブアプリのようにインストール・動作可能にする。
- **音声フィードバック:** カウント時に音を鳴らすことで、画面を見なくても練習に集中できるようにする。
